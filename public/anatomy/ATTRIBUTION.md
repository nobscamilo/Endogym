# Anatomy Atlas Attribution

The vector anatomy atlas used for the Endogym muscle activation map is derived from:

- **File:** Muscles front and back.svg
- **Source:** Wikimedia Commons
- **URL:** https://commons.wikimedia.org/wiki/File:Muscles_front_and_back.svg
- **Author:** Mikael Häggström, based on OpenStax College anatomy imagery
- **License:** Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)

Endogym generated grayscale base and alpha mask layers from that SVG for the muscle activation UI. Generated derivatives in `public/anatomy/vector-layers/` and `public/anatomy/vector-muscles-base.png` should preserve this attribution and license context.
